Interfaces to start your application with.

## Install

Using Composer:

~~~
$ composer require qlimix/kernel
~~~

## Testing
To run all unit tests locally with PHPUnit:

~~~
$ vendor/bin/phpunit
~~~

## Quality
To ensure code quality run grumphp which will run all tools:

~~~
$ vendor/bin/grumphp run
~~~
